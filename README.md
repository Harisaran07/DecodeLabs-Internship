# ✈️ JourneyNest | Premium Travel Booking Platform

A stunning, state-of-the-art interactive travel landing page built with premium **2025 aesthetics**, dynamic transitions, and modern design systems.

---

## ✨ Features

### 📅 1. 3D Calendar Page-Flip Navigation
*   An elegant, high-fidelity **3D page-flip effect** on the top navigation bar items (**Home, Destinations, Experiences, Contact, About**).
*   When clicked, the link text curls upwards and lifts forward like a sheet of paper being turned on a calendar, complete with realistic 3D perspective and shifting shadows.

### 🎨 2. Curated Color Palette & Matching Actions
*   Built on a harmonious, sophisticated theme consisting of **Mocha** (`#A1785F`) for stability/primary actions, **Ethereal Blue** (`#A9C4EB`) for secondary actions, and **Warm Greys** (`#F2F0EA`) for background refinement.
*   The **Sign In** button has been perfectly color-matched to the primary **Search** button, ensuring unified call-to-actions across the interface.

### 🌗 3. Interactive Light & Dark Modes
*   Features a responsive theme-switching system (using `localStorage` state retention).
*   Changes all layout variables, backgrounds, borders, card types, and typography colors smoothly to adapt to the user's preference.

### 🖼️ 4. Dynamic Hover Background Crossfade
*   Hovering over any popular destination card smoothly crossfades the section's background image with a blurred, parallax-simulated high-res preview of the selected spot, creating an immersive, alive experience.

---

## 📂 File Architecture

*   **`index.html`**: Structured semantic HTML layout supporting fluid responsiveness and grid-based galleries.
*   **`style.css`**: Pure vanilla CSS containing core color systems, typography tokens, media queries, and premium custom 3D keyframe animations.
*   **`script.js`**: Lightweight vanilla JavaScript handling page scroll states, tab switching, custom intersection observers for entrance animations, and click-animation state triggers.

---

## 🛠️ Technologies Used

*   **Structure**: HTML5 (Semantic elements)
*   **Styling**: CSS3 (Vanilla custom variables, 3D CSS Transforms, Keyframe Animations)
*   **Behavior**: Vanilla JavaScript (ES6+)
*   **Icons & Fonts**: Google Fonts (Montserrat & Roboto), FontAwesome 6
